🚀 **AI Visual Image Clustering Dashboard**

An intelligent AI-powered Image Clustering Dashboard that automatically groups visually similar images using deep learning embeddings and presents results through an interactive web interface.

This project combines Artificial Intelligence, Backend APIs, and Modern Frontend UI to deliver an end-to-end full-stack AI application.

---

📌 **Project Overview**

The AI Visual Image Clustering Dashboard allows users to upload multiple images and automatically organize them into clusters based on visual similarity.

Instead of manually sorting images, the system uses deep learning feature extraction and clustering algorithms to intelligently group related images.

---

✨ **Key Features**

✅ Upload multiple images
✅ Automatic AI-based image clustering
✅ Visual grouping of similar images
✅ Interactive dashboard interface
✅ Fast API backend processing
✅ Modern React frontend UI
✅ Scalable architecture for AI applications

---

🧠 **AI Workflow**

1. User uploads images through frontend
2. Backend processes images using AI model
3. Feature embeddings are generated
4. Clustering algorithm groups similar images
5. Results displayed visually on dashboard

⚙️ **Installation & Setup**

1️⃣ Clone Repository

git clone https://github.com/Nandhangit/AI-Visual-Image-Clustering-Dashboard.git
cd AI-Visual-Image-Clustering-Dashboard

---

2️⃣ Backend Setup

cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt

Run backend:

uvicorn api:app --reload

---

3️⃣ Frontend Setup

cd frontend
npm install
npm start
